5、克鲁斯卡尔(Kruskal)算法
（1）算法思想
　　①T的初始状态
   　   只有n个顶点而无边的森林T=(V，￠ )
　　②按边长递增的顺序选择E中的n-1安全边(u，v)并加入T，生成MST
注意：
    　安全边指两个端点分别是森林T里两棵树中的顶点的边。加入安全边，可将森林中的两棵树连接成一棵更大的树
    　因为每一次添加到T中的边均是当前权值最小的安全边，MST性质也能保证最终的T是一棵最小生成树。

（2）算法特点
    　该算法的特点是：当前形成的集合T除最后的结果外，始终是一个森林。

（3）Kruskal算法简单描述
T=(V, Φ ) ；
While (T 中所有边数 <n-1)
{ 从 E 中选取当前最短边 (u,v) ；
从 E 中删去边 (u,v) ；
If ((u,v) 并入 T 之后不产生回路 ) 将 (u,v) 并入 T 中；
}
